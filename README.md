# 🚲 Bike Sales Dashboard (Excel Project)

![Dashboard Screenshot](output.png)

## 📌 Overview

The **Bike Sales Dashboard** is an interactive data visualization project built using Microsoft Excel. It helps analyze customer purchasing behavior based on different factors like income, age, gender, region, and commute distance.

This dashboard allows users to filter data dynamically using slicers and gain meaningful business insights.

---

## 🎯 Objectives

* Analyze customer buying behavior
* Identify key factors affecting bike purchases
* Build an interactive and user-friendly dashboard
* Demonstrate data visualization skills using Excel

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning Techniques

---

## 📊 Dashboard Features

### 🔹 Interactive Slicers

* Marital Status
* Region
* Education
* Gender
* Children

These slicers allow users to filter data in real-time and update all charts dynamically.

---

### 🔹 Avg Income Per Purchase

* Compares average income of customers
* Segmented by gender and purchase behavior

**Insight:**
Customers with higher income are more likely to purchase bikes.

---

### 🔹 Customer Age Trend

* Shows distribution of customers across age groups
* Compares purchase vs non-purchase

**Insight:**
Middle-aged customers are the most active buyers.

---

### 🔹 Customer Commute Analysis

* Analyzes how commute distance impacts purchase decisions

**Insight:**
Customers with shorter commute distances tend to purchase more bikes.

---

## 📈 Key Insights

* Middle-aged customers contribute the highest sales
* Higher income increases purchase probability
* Short-distance commuters are more likely to buy bikes
* Customer behavior varies across regions

---

## 🧹 Data Cleaning Steps

* Converted abbreviations (F/M → Female/Male)
* Removed duplicates
* Handled missing values
* Standardized categorical data

---

## ⚙️ How It Works

1. User selects filters using slicers
2. Pivot tables update automatically
3. Charts reflect filtered data
4. Insights change dynamically

---
